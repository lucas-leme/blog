Introdução

Quanto tempo de vida um paciente ainda viverá? Qual será a temperatura amanhã? Qual será o preço do dólar daqui a 1 mês? Para responder essas perguntas podemos realizar uma análise fundamentada e chutar um valor, 1 ano, 30 graus e 5 reais, respectivamente. Mas, será que essas respostas são suficientes? Há muita incerteza assoaciada a previsões, e é importante que as previsões sejam capazes de capturar essa incerteza.

Então o ideal é que as previsões sejam probabilísticas, ou seja, que elas sejam capazes de dizer que a temperatura amanhã estará entre 25 e 35 graus com 95% de confiança, por exemplo. Existem alguns métodos para fazer previsões probabilísticas, como Monte Carlo e Métodos Bayesianos, mas esses métodos são computacionalmente caros e exigem um maior trabalho de modelagem. Visando resolver esse problema, os autores Duan et al. (2020) propuseram o algoritmo NGBoost, que é um algoritmo de boosting que faz previsões probabilísticas de maneira simples e eficiente.

Como funciona o NGBoost?

NGBoost é um algoritmo de regressão probabilística, que dada uma distribuição paramétrica e um modelo de regressão, estima os parâmetros da distribuição de probabilidade a partir dos dados de entrada do modelo de regressão. Então para o caso de uma distribuição normal, o algoritmo estima os parâmetros: média e o desvio padrão, então a partir dos dados de entrada \(x\) o modelo irá estimar as funções \(\mu(x)\) e \(\sigma(x)\). Assim, o algoritmo é capaz de fazer previsões probabilísticas e sem assumir homecedasticidade dados!

Para encontrar os parâmetros da distribuição os autores utilizam um algoritmo de boosting denominado Natural Gradient Boosting, que é uma variação do Gradient Boosting, que ao invés de encontrar o ponto mínimo da função num espaço de distância euclidiana, encontra o ponto mínimo da função num espaço transformado que é apropriado para distribuições de probabilidades. Portanto, a convergência do gradiente natural é mais rápida (Figura 1) visto que as características das distribuições de probabilidades são levadas em conta.

Como funciona um algoritmo de boosting?

Um algoritmo de boosting utiliza uma técnica de ensemble learning, que consiste em combinar a previsão de diversos modelos para tomar a decisão final. No caso do boosting, os modelos são treinados sequencialmente, onde cada modelo tenta corrigir os erros do modelo anterior. Então, o primeiro modelo é treinado e faz suas previsões, em seguida o segundo modelo é treinado para corrigir os erros do primeiro modelo e assim por diante como mostra a Figura 2.

Então no processo de aprendizado do NGBoost, para cada parâmetro de um distribuição de probabilidade são treinados diversos modelos de maneira sequencial, onde cada modelo tenta corrigir os erros do modelo anterior.

Os algoritmos de Boosting funcionam muito bem com modelos de árvore de decisão, então por padrão o NGBoost utiliza árvores de decisão como modelo base, mas um diferencial da implementação em Python da biblioteca NGBoost é que é possível utilizar qualquer modelo de regressão como modelo base, como por exemplo, regressão linear, regressão logística, SVM e entre outros.
